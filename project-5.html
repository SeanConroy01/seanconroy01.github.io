Gecko Game

Project 05 - Be the gecko and avoid the branches!

Avoiding obstacles has never been so addicting! Be the gecko and avoid the branches! How long will you survive, and how many skins will you unlock?

The concept behind The Gecko Game was to develop an iOS application inspired by the game Flappy Birds. The game would use a similar art style, character design and game mechanic. However, instead of bird bouncing to avoid pipes, it would use a Gecko that would flip around a branch to avoid further oncoming branches. The user would then unlock different skins as they progressed and reached predefined milestones to have some player progression. The application was developed in Swift and took approximately three weeks to develop.
