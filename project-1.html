UpComing

Project 01 - Countdown to the moments that matter

It's the experiences in life that matter most. UpComing helps you remember and count down to all those special moments. Whether that be an anniversary, birthday, concert, holiday or wedding, see how many years, months, weeks, days, hours, minutes, and seconds they are away.

The concept behind UpComing was to develop an iOS application that would allow individuals to track how long they had until their various upcoming events. The application needed to have a better user interface than the many existing apps already on the App Store. It could also include additional features to change the events background colour, add custom images, a dark mode, and the ability to change the units of time displayed. The application was developed in Swift using the model view controller (MVC) design pattern and took approximately two months to develop. The application has been temporarily removed for the App Store to give me time to add some additional features and fix some identified issues.
